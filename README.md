# harmony eth bridge frontend
Ethereum<>Harmony two way bridge (trusted version)



## Install instructions

### Requirements 

* nodejs 

### Commands

* Fetch repo 

```
git clone https://github.com/harmony-one/layerzero-bridge.frontend
```

* Install dependencies

```
npm install
```

* Develop

```
npm run dev
```

* Build

```
npm run build
```

* Start

```
cd build 
serve
```
